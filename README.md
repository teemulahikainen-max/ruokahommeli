# Arkiruoka

Kevyt selainpohjainen sovellus viikoittaiseen ruokasuunnitteluun.

## Sisältö

- ruokapankki omille ruoille
- viikonäkymä lounaalle ja päivälliselle
- automaattinen täydennys tyhjille kohdille
- kauppalista viikon valinnoista
- tiedot tallennetaan selaimen localStorageen

## Käyttö

Avaa `index.html` selaimessa.
